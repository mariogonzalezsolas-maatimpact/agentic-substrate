# Error Log - Feedback Loop

Persistent log of mistakes made during sessions. Claude reads this at session start and never repeats a documented error.

## Format

Each entry follows this structure:

```
### [YYYY-MM-DD] Error Title
- **What happened**: Brief description of the mistake
- **Root cause**: Why it happened
- **Correct approach**: What should have been done instead
- **Prevention**: Rule to follow going forward
```
